# Adarsh Portfolio

Personal portfolio website built by Adarsh Kumar Priyadarshi.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- GSAP

## Getting Started

```sh
git clone https://github.com/AdarshKumarPriyadarshi/Adarsh-Portfolio.git
cd Adarsh-Portfolio
npm install
npm run dev
```

## Deployment

The site is deployed to GitHub Pages via GitHub Actions.
